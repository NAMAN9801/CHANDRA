About Project Chandra

Project CHANDRA (Critical Hyperspectral Analysis of Naturally Dark Remote Craters) is an advanced imaging analysis system designed for crater detection and analysis using PSR (Permanently Shadowed Region) detection.

How to Use:

  • Upload an Image: Click on "Choose an Image" to upload your own crater image for analysis.
  • Select Test Images: Or choose from our curated test images to see immediate results.
  • Processing: The system will analyze your image and display real-time progress with a timer.
  • View Results: Analyze 6 different visualizations including original, enhanced, PSR detection, edges, and roughness analysis.
  • Download: Export your complete analysis as a high-quality image.

Analysis Methods:

  • CLAHE Enhancement: Improves contrast using adaptive histogram equalization.
  • Threshold PSR: Basic threshold-based PSR region detection.
  • Adaptive PSR: Advanced adaptive threshold approach for better accuracy.
  • Edge Detection: Identifies crater boundaries using Canny edge detection.
  • Roughness Analysis: Maps surface terrain variations and roughness.
